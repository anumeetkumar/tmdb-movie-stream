Developer Integration Guide

TMDB Embed Player provides an easy, zero-setup HLS streaming embed interface. Integrate it in seconds.

HTML Iframe Embed API

To embed movies or episodes seamlessly in your streaming portal, simply load our responsive URL endpoints in an iframe element.

Routing URL format Description
https://your-domain.com/embed/movie/{tmdb_id} Embeds a movie player matching the specified TMDB movie database identifier.
https://your-domain.com/embed/tv/{tmdb_id}/{season}/{episode} Embeds a specific television episode player with season and episode coordinates.

Query String Options

Option Allowed Values Default Description
embed true, false true on /embed path Enables absolute fullscreen view (hides header/footer layout).
proxy true, false true Determines whether streams are proxy-rewritten or direct.

REST JSON Stream Resolvers

For professional platforms building fully custom players, fetch the direct HLS `.m3u8` feeds using our REST endpoints.

Smart Failovers & Dual-Phase Resolution

TMDB Embed Player is engineered for zero buffering. We call multiple scrapers concurrently and update sources dynamically.

Dual-Phase Player Loading

Our player doesn't block loading on slow resolvers. It loads the direct Vidzee Hindi fast-stream instantly (<500ms), and then populates secondary backup streams (DahmerMovies, Nxsha, etc.) in the background when resolved.

Unproxied Fast Feeds

Unlike other embeds that proxy all streams (causing slow starts and Vercel proxy limits), we identify clean direct CDNs (like Vidzee) and stream them completely unproxied for high speeds.
